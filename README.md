## Info

- This project saves data in memory so you don't need any database
- This project uses [NPM Workspaces](https://docs.npmjs.com/cli/v7/using-npm/workspaces)

## Requirements

- NodeJS (16.x.x)
- NPM (8.x.x)

## Start the application

1. Install dependencies

```
npm i
```

2. Start backend

```
npm start -w server
```

3. Start client

```
npm start -w client
```
